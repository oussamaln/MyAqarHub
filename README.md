# 🏠 MyAqarHub

> The modern real estate and investment platform for Annaba, Algeria.

## 🌍 Live Demo

https://myaqarhub-ga3x5dnh.manus.space/

## ✨ Features

- 🏗️ Off-plan real estate projects
- 🏘️ Buy & Sell properties
- 🛏️ Rental listings
- 👷 Verified developers
- 📈 Investment opportunities
- 🗺️ Interactive map
- 📊 Construction progress tracking

## 🚀 Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
- Drizzle ORM
- Node.js
